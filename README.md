# Student Management System

## Project Overview
A comprehensive C++ Student Management System demonstrating Object-Oriented Programming principles, file persistence, and professional software development practices. This console application allows educational institutions to efficiently manage student records with full CRUD (Create, Read, Update, Delete) operations.

## Features
- ✅ **Add Student**: Register new students with roll number, name, age, grade, and email
- ✅ **Delete Student**: Remove student records by roll number
- ✅ **Search Student**: Find students by roll number with detailed display
- ✅ **Update Information**: Modify existing student details with validation
- ✅ **Display All**: View all students in formatted table
- ✅ **Statistics**: View system statistics (total students, average age, age range)
- ✅ **Data Persistence**: Automatic save/load to students.txt file
- ✅ **Input Validation**: Comprehensive error handling for all inputs
- ✅ **OOP Design**: Proper encapsulation, inheritance-ready structure
- ✅ **STL Usage**: Vector for dynamic storage, algorithms for sorting
